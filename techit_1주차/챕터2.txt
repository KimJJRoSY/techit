
#쿠키 = 서버->클라이언트 보내지는 아주 작은 데이터 
	=키/값 구조 ->유효기간 있음 

쿠키가 없으면  서버이동 할때마다 로그인 해야됨 

#세션 저장소 

#네트워크 - 두 대 이상의 컴퓨터가 연결된 통신망

#라우터 - 네트워크를 구성하는 스위치의 역할 =>즉 공유기 

#IP-컴퓨터 간 데이터를 주고받는 네트워크 계층의 규약  
	-데이터 전달에 필요한 목적지 컴퓨터 정보가 필요 

#IP주소 -네트워크에서 컴퓨터가 부여받는 고유의 주소 

#IP4 -32비트의 용량 가질 수 있음 

#IP6 - 128비트 ->완전히 자리 잡으건 아님 ip4다음에 나옴 

#공인IP - 전체 인터넷 망에서 고유하게 식별 가능한 주소 
	     -IP4체계에서 자원 부족 
	      -하나의 공인 ip에서 수많은 사설 ip 할당 가능

#사설IP - 가정의 lan과 같은 네트워크에서 할당되는 주소
	     -컴퓨터에서 조회되는 IP 

#ISP -인터넷 서비스 제공자 
	-공인 ip 할당해줌 

#127.0.0.1 -localhost =>내가 사용하는 내 컴퓨터에서 사용되는 내 컴퓨터의 별명
	
#PORT
하나의 컴퓨터에는 다양한 서비스가 실행중 
->서비스를 구분하는 역할 필요 => 포트 

#DNS
-URL을 해석해 ip주소로 반환하는 서버
-국가, 기업 등이 운영 
-전세계 DNS는 연결되어 있음 => 장애 발생 시 막대한 피해로 이어짐
